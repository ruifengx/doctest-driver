declEntity :: HsDecl GhcPs -> Maybe Entity Source #

Extract an Entity for the declaration.

This is used to determine which declaration a Haddock document should attach to.

  • Just e if at least one entity is found, and e is the first entity.
  • Nothing if there is no entity the document could attach to.

extractModule :: ParsedModule -> Module Source #

Extract a Module from a GHC ParsedModule.

Implementation Note

The documentation in a ParsedModule is first extracted into a doc-tree (via collectDocs). By design, doctest instructions can appear as group names (Haddock markup $group-name) and as comments in verbatim code block (Haddock markup @...@). Inside this doc-tree, we inspect these places, recognise the instructions, and apply the instructions. Some instructions modify their parent scope (Captures and IOHooks bring new names in scope), so we float them above.
